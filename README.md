# currency-converter-app
<p>A modern and responsive Currency Converter web application built using HTML, CSS, and JavaScript.</p>

<h4>This app allows users to:</h4>

 <p>Convert currencies in real-time using live exchange rates</p>
 <p>Swap currencies instantly</p>
 <p>Auto-update country flags based on selected currency</p>
 <p>Press Enter to convert quickly</p>
 <p>Fetch live exchange rates using ExchangeRate API</p>
